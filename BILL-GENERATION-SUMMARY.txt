QUICK START TESTING:
================================================================================

1. End a completed session:
   POST /api/sessions/{SESSION_ID}/end

2. Generate bill from session:
   POST /api/sessions/{SESSION_ID}/generate-bill
   {
     "items": [
       {
         "itemName": "Energy Drink",
         "quantity": 2,
         "unitPrice": 2.50
       }
     ],
     "discount": 1.50,
     "taxRate": 0.10,
     "paymentMethod": "card"
   }

3. Mark bill as paid:
   POST /api/bills/{BILL_ID}/pay
   {
     "paymentMethod": "card"
   }

================================================================================

HOW IT WORKS:
================================================================================

Session Running → End Session → Generate Bill → Mark as Paid

When you end a session:
  • Session status → "completed"
  • Duration & cost calculated automatically
  • Session marked for bill generation

When you generate bill:
  • Bill created with session cost
  • Optional items added with pricing
  • Discount applied
  • Tax calculated automatically
  • Total amount calculated
  • Bill linked to session
  • Session marked as billGenerated: true

When you mark paid:
  • Bill status → "paid"
  • Payment method recorded
  • Transaction complete

================================================================================

BILL CALCULATION FORMULA:
================================================================================

subtotal = sessionCost + itemsTotal
tax = subtotal × taxRate
total = subtotal + tax - discount

EXAMPLE:
─────────────────────────────────────
Session Cost: $225.00
Items Total: $15.50
────────────────────────────────────
Subtotal: $240.50
Tax (10%): $24.05
Discount: ($5.00)
════════════════════════════════════
TOTAL: $259.55

ERROR HANDLING:
================================================================================

400 - Session must be completed before generating bill
      → Call /end endpoint first

404 - Session not found
      → Verify correct session ID

400 - Bill already exists for this session
      → Use existing bill ID or create new session
