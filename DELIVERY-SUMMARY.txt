STEP-BY-STEP GETTING STARTED
================================================================================

STEP 1: Download/Extract
   - Download the project from v0 or GitHub
   - Extract to your desired location

STEP 2: Read START-HERE.md
   - Open START-HERE.md
   - Decide which guide to follow based on your needs

STEP 3: Install Dependencies
   $ cd gaming-cafe-system
   $ pnpm install --recursive

STEP 4: Setup MongoDB
   - Install MongoDB locally OR
   - Create MongoDB Atlas account (free tier)

STEP 5: Configure Environment
   - Update backend/.env with MongoDB connection
   - Update frontend/.env if needed

STEP 6: Start the Application
   $ pnpm dev
   - This starts both frontend AND backend

STEP 7: Open Browser
   - Frontend: http://localhost:5173
   - Backend: http://localhost:5000

STEP 8: Register/Login
   - Register new admin OR
   - Login with test credentials above

STEP 9: Test APIs (Optional but Recommended)
   - Import Gaming-Cafe-API.postman_collection.json
   - Follow POSTMAN-TESTING-GUIDE.md

STEP 10: Explore!
   - Use the dashboard
   - Create plans, stations, sessions
   - Generate bills
   - View reports

COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "Cannot find module 'express'"
Solution: Run $ pnpm install --recursive

Issue: "Connection refused - localhost:5000"
Solution: Backend is not running. Run $ pnpm dev --workspace=backend

Issue: "MongoDB connection failed"
Solution: Start MongoDB or check connection string in .env

Issue: "Port 5000 already in use"
Solution: Kill the process or change PORT in .env

Issue: "Download button not working in v0"
Solution: Use GitHub to clone, or contact support

See DOWNLOAD-AND-SETUP.md for complete troubleshooting section
